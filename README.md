# 15% Discount Voucher - Perla Homes

A promotional web page showcasing **15% discount vouchers** for the Greek restaurant *Therapeutirion Tsipouradiko*.  
Built with **HTML and CSS**, it is fully responsive and includes a TripAdvisor link, map location, and footer with Perla Homes branding.
